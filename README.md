# lab-08-many-to-many

### Setup

1. Create repo, clone, and open vscode
2. npm init @alchemycodelab/app@latest (sql only)
3. npm i express
4. npm i -D supertest
5. git checkout -b dev

## Requirements

- CRUD routes for your model (2 points)
- At least one join (5 points)
- test CRUD routes (3 points)
